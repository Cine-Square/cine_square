## 주요기능

- 로그인, 회원가입
- 영화 상세페이지 별점 및 댓글
- 영화검색 및 조회
- 프로필 페이지

## 설치 및 실행

1. 설치

```
$ git clone 'https://github.com/Cine-Square/cine_square.git'
$ npm install
```

2. 실행

```
$ npm start
```
